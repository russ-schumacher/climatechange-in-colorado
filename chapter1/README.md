Excel file contains global, U.S. and Colorado annual temperatures from 1900 - 2021.
Temperature departures are calculated as the difference from the 20th century average.

All data in Excel file are publicly available via the NOAA NCEI Climate At A Glance website:
https://www.ncei.noaa.gov/access/monitoring/climate-at-a-glance/

Figure 1.1 is charted as a separate sheet in the Excel file.
Five year rolling average temperature departures are graphed compared with the 20th century average denoted with a dashed black line.
